# Arxiv papers map

Build a map of citations for arxiv papers. [[Demo](http://fabiendelattre.com/arxivpapersmap/)]


![](demo.gif)


Run `npm install` to install the dependencies.
Feel free to contribute and to propose ideas in the issues.
